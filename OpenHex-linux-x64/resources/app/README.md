# OpenHex electron builder

Build OpenHex application and package it to an electron application.


## Install

``` bash
git clone git@github.com:alcalyn/openhex-electron.git
cd openhex-electron/

npm install
```


## Usage

Test OpenHex:

``` bash
npm start
```

Build OpenHex for your platform:

``` bash
npm run build
```
